# 🌸 Lilac Store — Flower Shop Management System

A full-stack **Client/Server application** for managing a flower shop, built with Java as a Software Engineering team project.

---

## 📌 Overview

Lilac Store is a comprehensive flower shop management system that handles:
- 🌺 **Inventory Management** — track flower stock in real time
- 📦 **Order Processing** — manage customer orders end-to-end
- 👥 **Customer Interaction** — smooth client-facing interface
- 🔄 **Client/Server Architecture** — robust server handling multiple clients

## 🚀 Getting Started

### Prerequisites
- Java JDK 11+
- Maven 3.6+
- IntelliJ IDEA (recommended)

### Installation

```bash
# Clone the repository
git clone https://github.com/mohammed761/Lilac-store-SV.git
cd Lilac-store-SV
```

### Run the Server
```bash
cd Server-master
mvn clean install
mvn exec:java
```

### Run the Client
```bash
cd Client-master
mvn clean install
mvn exec:java
```
